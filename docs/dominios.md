## Reglas de asignación

1. Un dominio por aprendiz por trimestre
2. No repetir dominio entre aprendices del mismo grupo
3. Los dominios `★` están reservados solo para ejemplos del bootcamp
4. Al cambiar de trimestre, los dominios quedan disponibles nuevamente (salvo re-inscripción)
