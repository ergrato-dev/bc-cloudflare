# Semana 09 — Hyperdrive y Bases de Datos Externas

> **Etapa 1 — Persistencia y Datos** · Bootcamp Cloudflare De Cero a Héroe

---

## 🎯 Objetivos

1. Entender por qué los Workers no pueden conectarse directamente a PostgreSQL y cómo Hyperdrive resuelve el problema
2. Configurar un Hyperdrive config en `wrangler.jsonc` y obtener el connection string de pooling
3. Ejecutar consultas SQL type-safe desde un Worker usando `postgres.js`
4. Activar el caching de queries de Hyperdrive para reducir latencia en lecturas frecuentes
